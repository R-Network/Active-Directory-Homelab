# Advanced Audit Policy

Advanced Audit Policy is configured in the `Company Workstation Baseline` GPO to collect workstation security events for authentication, object access, privilege use, policy changes, and system activity.

## Implemented Audit Areas

- Logon and logoff auditing.
- Account lockout auditing.
- Credential validation auditing.
- Process creation auditing.
- Directory service access auditing.
- File share auditing.
- Policy change auditing.
- Privilege use auditing.
- System integrity auditing.

## Benefits

- Supports incident response.
- Helps identify suspicious activity.
- Provides evidence for security troubleshooting.
- Improves visibility into workstation behavior.
